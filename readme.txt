Данный репозиторий содержит базовый набор инструментов и переведённый на русский
язык текст игры Adventure Time: The Secret of the Nameless Kingdom.

- fonts - шрифты в формате TTF/OTF с добавленными пикторграммами кнопок соответствующей платформы 
- logo_covers - папка с русским логотипом в высоком разрешении и пара кастомных обложек в т.ч. в .PSD
- ltbtools - скрипт на питоне для выемки/вставки текста из лангпака localization.ltb
- text - переведённый на русский язык текст для 3DS/PSV. Внимание! Количество строк для разных платформ отличается.
