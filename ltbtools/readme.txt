ltbtools
WayForward Engine text resource packer

Требуется python 2

Отличия:
3DS - 779 строк в ланпаке для каждого языка
PSV - 820 строк в ланпаке для каждого языка

По умолчанию вынимает и вставляет текст из/вместо внглийских строк.

Оффсет указателей прописан в строке #60
Количество вынимаемых строк указано в строке #62
